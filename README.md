# 🚀 GitHub Actions - Быстрая инструкция

## 📋 ШАГ 1: Создайте репозиторий

1. Откройте https://github.com/new
2. Название: `ocbk-android`
3. Public ✅
4. Create repository

---

## 📋 ШАГ 2: Загрузите файлы

Нажмите **"uploading an existing file"** и загрузите:

- ✅ main.py
- ✅ ocbk_app.py
- ✅ russian_dictionary.txt
- ✅ buildozer.spec
- ✅ .github/workflows/build.yml

Нажмите **"Commit changes"**

---

## 📋 ШАГ 3: Запустите сборку

1. Вкладка **Actions**
2. Нажмите **"Build Android APK"**
3. Нажмите **"Run workflow"**

---

## 📋 ШАГ 4: Ждите 20-30 минут

---

## 📋 ШАГ 5: Скачайте APK

**Actions** → Клик на запуск → **Artifacts** → **OCBK-APK**

---

## 📱 Установка на телефон

1. Распакуйте ZIP
2. Скопируйте APK на телефон
3. Установите

---

**Готово! 🎉**
